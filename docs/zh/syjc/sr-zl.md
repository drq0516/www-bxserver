# 崩坏：星穹铁道指令教程

## 感谢您的支持

 1. 将指令生成网站链接**打开**
      [https://cmd.bxgov.cn:21460](https://cmd.bxgov.cn:21460)

 2. 打开后**如下**所示

 ![bd](/images/sr-zl/cbb6417644f69d720772eec3ebdd671f.jpg)

 3. 点击**开始**，进入指令生成界面，点击**复制**指令

  以下以星琼获取指令为例，其他功能还请自己尝试，~~规则就是用来打破的~~
